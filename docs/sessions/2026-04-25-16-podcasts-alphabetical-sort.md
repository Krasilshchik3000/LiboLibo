# Сессия 16 — алфавитная сортировка подкастов

**Дата:** 2026-04-25
**Участники:** Илья, Claude (Opus 4.7)
**Контекст:** на экране «Подкасты» три секции — «Выходят сейчас», «Недавно выходили», «Давно не выходят». До этой сессии внутри каждой секции подкасты были отсортированы по дате последнего эпизода (по убыванию). Илья попросил поменять сортировку на алфавитную внутри каждой секции — состав секций при этом сохраняется.

## Что сделали

В [PodcastsView.swift](../../LiboLibo/Features/Podcasts/PodcastsView.swift) у трёх вычисляемых свойств (`active`, `recent`, `dormant`) заменили `.sorted` по `lastEpisodeDate` на сортировку по `name` через `localizedCaseInsensitiveCompare` — корректно работает для русских строк и регистронезависимо.

Логика отнесения подкастов к секциям не менялась: «выходят сейчас» — последний эпизод за последние 90 дней, «недавно» — 90–365 дней, «давно не выходят» — старше 365 дней или вообще без эпизодов.

## Новое правило (введено в этой сессии)

Илья попросил, чтобы каждая задача в LiboLibo заканчивалась полным циклом: новая сборка → установка в симулятор → коммит и пуш в GitHub. Старое правило в [CLAUDE.md](../../CLAUDE.md) «не делай коммитов без явной просьбы» снято: теперь сама просьба сделать задачу = просьба запушить результат. Зафиксировано в новом разделе «Ритуал завершения задачи (iOS)» в `CLAUDE.md` и в auto-memory.
